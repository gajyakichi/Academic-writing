# Data ディレクトリ

このディレクトリは、論文の表作成に使用する CSV/Excel ファイルを格納します。

## ファイル一覧

### サンプルファイル

- `sample_baseline.csv` - 患者の基本特性（サンプル）
- `sample_multivariate.csv` - 多変量解析の結果（サンプル）

## 使い方

### 1. データファイルの配置

CSV ファイルまたは Excel ファイルをこのディレクトリに配置します。

### 2. LaTeX 表への変換

```bash
# CSVファイルを変換
python3 csv_to_latex.py data/your_file.csv \
  -c "表のタイトル" \
  -l "tab:your_label"

# 出力をファイルに保存
python3 csv_to_latex.py data/your_file.csv \
  -c "表のタイトル" \
  -l "tab:your_label" \
  -o tables/your_table.tex
```

### 3. 論文への組み込み

```latex
% main_J.tex または main.tex
\input{tables/your_table.tex}

% または直接コピー&ペースト
```

## CSV ファイルの形式

### 基本的な形式

```csv
列名1,列名2,列名3
データ1,データ2,データ3
データ4,データ5,データ6
```

### 例: 患者の基本特性

```csv
変数,対照群,治療群,p値
年齢（歳）,65.2 ± 8.3,64.8 ± 7.9,0.75
男性（%）,60.0,58.0,0.82
BMI (kg/m²),24.5 ± 3.2,24.1 ± 3.0,0.58
```

## 重要な注意事項

1. **エンコーディング**: ファイルは UTF-8 で保存してください
2. **カンマ区切り**: データはカンマ（,）で区切ってください
3. **ヘッダー行**: 最初の行は列名として扱われます
4. **特殊文字**: LaTeX の特殊文字（%, &, $など）は自動的にエスケープされます

## Excel からの保存方法

1. Excel でデータを作成
2. 「ファイル」→「名前を付けて保存」
3. ファイル形式で「CSV UTF-8（カンマ区切り）（\*.csv）」を選択
4. このディレクトリ（`data/`）に保存
